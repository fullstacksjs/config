# Getting Started

## Installation

### Node

npm

```shell copy
npm install --save-dev @fullstacksjs/config
```

pnpm

```shell copy
pnpm add --save-dev @fullstacksjs/config
```

yarn

```shell copy
yarn add --dev @fullstacksjs/config
```

### Deno

```typescript copy
import { Config } from 'https://raw.githubusercontent.com/fullstacksjs/config/main/mod.ts';
```

### Browser ESM

ES Module

```html copy
<script type="module">
  import { Config } from 'https://www.unpkg.com/@fullstacksjs/config@<VERSION>/lib/esm/index.js';
</script>
```

Script

```html copy
<script src="https://www.unpkg.com/@fullstacksjs/config/iife/index.js"></script>
<script>
  console.log(window.Config);
</script>
```
